# Min Portfolio

En personlig portfolio byggd med HTML, CSS, JavaScript och Vite.

## Filer

- `index.html` – sidans innehåll och struktur
- `src/style.css` – färger, layout och mobilanpassning
- `src/main.js` – mobilmeny och animationer
- `src/assets/bild1.jpg` – profilbild
- `package.json` – projektets kommandon och Vite

## Starta projektet

```bash
npm install
npm run dev
```

Skapa en färdig produktionsversion med:

```bash
npm run build
```
